# Quick Start: Font Customization ⚡

## For End Users (Non-Technical)

### What You Can Do Now
You can change the font and text size of your data table - **no coding required!**

---

## 🎯 In 3 Simple Steps:

### Step 1: Open Extension Settings
When you add or edit the "Custom Data Table" control on your work item, you'll see these NEW fields:

```
┌─────────────────────────────────────────┐
│ Font Family                             │
│ ┌─────────────────────────────────────┐ │
│ │ [Type font name here]               │ │
│ └─────────────────────────────────────┘ │
│ Choose a font family for the table.     │
│ Leave empty for default.                │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ Font Size (pixels)                      │
│ ┌─────────────────────────────────────┐ │
│ │ [Type size here]                    │ │
│ └─────────────────────────────────────┘ │
│ Set font size in pixels (e.g., 12, 14,  │
│ 16). Leave empty for default (13px).    │
└─────────────────────────────────────────┘
```

### Step 2: Choose Your Settings

**Popular Choices:**

| What You Want | Font Family | Font Size |
|---------------|-------------|-----------|
| **Larger text (easier to read)** | Arial | 16 |
| **Smaller text (fit more)** | Verdana | 11 |
| **Professional look** | Segoe UI | 14 |
| **Classic style** | Georgia | 14 |
| **Keep default** | *(leave empty)* | *(leave empty)* |

### Step 3: Save and Enjoy!
Click Save → Your table now uses your chosen font! 🎉

---

## 💡 Tips

### Recommended Font Names:
Just type one of these (exactly as shown):
- `Arial`
- `Verdana` 
- `Tahoma`
- `Georgia`
- `Segoe UI`
- `Calibri`
- `Trebuchet MS`

### Recommended Font Sizes:
Just type the number:
- `11` = Small
- `12` = Bit smaller
- `13` = Default (if you leave empty)
- `14` = Bit larger
- `16` = Large
- `18` = Extra large

## ❓ Common Questions

**Q: What happens if I leave it empty?**
A: The table uses the default font (system font at 13px). This is perfectly fine!

**Q: Can I use any font?**
A: Stick to common fonts listed above. Custom fonts might not work for all users.

**Q: What if I make a mistake?**
A: No worries! Just leave both fields empty to go back to default.

**Q: Will this affect my existing data?**
A: No! This only changes how text looks, not your data.

**Q: Do other people see my font choice?**
A: Yes! Once you save the work item form layout, everyone using that work item type will see the same font.

## 📞 Need Help?

If something doesn't work:
1. Try leaving both fields empty (use defaults)
2. Make sure you spelled the font name correctly
3. Use a number between 10-20 for font size
4. Save and refresh your browser
